display: inline  기본값으로, 요소를 inline 요소처럼 표시합니다.
                    따라서 앞뒤로 줄바꿈 되지 않습니다.
 
display: block  요소를 block 요소처럼 표시합니다.
                    따라서 요소 앞 뒤로 줄바꿈 됩니다
 
display: none 박스가 생성되지 않습니다.
                   따라서 공간을 차지하지도 않습니다.
 
display: inline-block 요소는 inline인데 내부는 block 처럼 표시함. 

                                  즉, 박스 모양이 inline 처럼 옆으로 늘어섬.
