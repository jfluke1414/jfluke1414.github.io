title: (Javascript) String을 Number타입으로 바꾸기   
category: Javascript   
---   
   
```   
1	// 숫자를 스트링로 바꾸기   
2	var tt = 2   
3	tt += "";   
4	alert(typeof tt);   // Result : string   
5	    
6	// 스트링을 숫자로 바꾸기   
7	tt = "2"   
8	tt *= 1;   
9	alert(typeof tt);    // Result : number   
```
